# GroupoMania 

GroupoMania is a corporate social network project

### Install dependencies
this project use [***workspaces***](https://docs.npmjs.com/cli/v7/using-npm/workspaces) use ***npm*** or ***yarn*** to install dependencies
 
```bash
  npm install
```

### Run dev server

#### back-end server

```bash
  npm serve:back
```

#### front-end server

```bash
  npm serve:front
```
